# 🧠 Prompt Templates

Copy-paste, modify, and use these templates in your daily workflow.

## How to Use

1. Pick a template
2. Replace placeholders (`<...>`)
3. Add your code/context
4. Iterate on the output

## Principles

* Be explicit with instructions
* Always add constraints
* Provide context
* Never trust output blindly
